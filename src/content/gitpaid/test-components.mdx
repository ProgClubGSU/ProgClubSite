## Nested Components

<Card variant="bordered" title="Advanced React Project Example">

<CodeBlock
  code={`import React, { useState, useEffect } from 'react';

function UserDashboard() {
  const [users, setUsers] = useState([]);

  useEffect(() => {
    fetch('/api/users')
      .then(res => res.json())
      .then(data => setUsers(data));
  }, []);

  return (
    <div className="dashboard">
      {users.map(user => (
        <UserCard key={user.id} user={user} />
      ))}
    </div>
  );
}`}
  language="javascript"
  title="React Dashboard Component"
  showLineNumbers={true}
  client:load
/>

<Callout type="success" title="Great Job!">

### Quick Reference

**Static Components (No JavaScript overhead):**
- 🔔 **Callout** - Info boxes with 5 variants
- 🃏 **Card** - Content containers with 3 styles

**Interactive Components (React):**
- 🔘 **Button** - Fancy buttons with 6 variants, icons, links
- 📦 **ButtonGroup** - Group buttons with alignment options
- 📂 **Accordion** - Collapsible content sections
- 💻 **CodeBlock** - Syntax highlighted code with copy button
- 📅 **Timeline** - Visual progress timelines
- 🔢 **StepList** - Numbered step-by-step instructions
